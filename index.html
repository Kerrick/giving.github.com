I'm a scientist, engineer or charity ↓

How do I ask for help?

  1. formulate your project so that the problem you are asking for help with can be described clearly and completely in a paragraph or two
  2. re-read your project brief; is it really unambiguous what you want help with?
  3. all data must be yours to share, and help received is MIT licensed
  4. unless you explicitly state that you want or accept Affero GPL licensed code
  5. submit your brief as an 'issue' using the big blue button below ↓
  6. you'll be prompted to create a github account if you don't already have one
  7. if you have data to crunch, can you attach a simple small test-set to the issue?
  8. worried that your problem is open-ended, badly defined or too big? Perhaps the first project you post is asking for help in splitting it into smaller tasks

I'm a programmer ↓

5 minutes or an hour or an evening of your time is all you need spare to achieve something meaningful! You don't need dedication and commitment, only commit or code-review

Show me how to help!

  1. you need a github account and to be familiar with git/github
  2. browse the open issues (big blue button below ↓)
  3. all code given is marked clearly and properly as MIT licensed/Affero GPL licensed please
  • you may be able to contribute a solution or part of a solution or such; create a github public repo and code away; link to it in an issue comment
  • you may fork someone else's solution to make changes; send them a pull-request and comment in the issue
  • you may be able to take five minutes to look over some code someone has submitted; spot a bug? comment on the issue
  • you may have insights to share; comment on the issue!
